Installation
************

Installation
############

Install with `pip`:

::

    $ sudo apt-get install libffi-dev libssl-dev python3-dev
    $ pip3 install bitshares

Manual installation:

::

    $ git clone https://github.com/xeroc/python-bitshares/
    $ cd python-bitshares
    $ python3 setup.py install --user

Upgrade
#######

::

   $ pip3 install bitshares --user --upgrade
